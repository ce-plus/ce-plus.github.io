Features

Everything you need to write code

Built to stay out of your way. Clean interface, fast startup, and the tools developers actually use.

Syntax Highlighting
Rich color syntax for 18+ programming languages powered by CodeEditorLib. Automatic detection based on file extension.
Built-in Code Runner
Run Python, JavaScript, C/C++, Ruby, Java, and HTML directly from the editor. Output appears in the integrated panel below.
Multi-Tab Editing
Work with multiple files at once in a clean tab interface. Each tab tracks unsaved changes and supports drag-and-drop file opening.
7 Built-in Themes
Choose from 7 beautiful built-in themes, including light, dark, and colorful styles. Your selected theme is saved automatically for the next launch.
Code Minimap
Navigate large files easily with the built-in minimap. Toggle it on or off from the View menu. Line numbers can also be toggled.
Multi-Language UI
Interface available in English and Indonesian, with more languages easily added via simple INI files. Community translations welcome.

Open Source

Built on CodeEditorLib

CE++ is fully open source, and so is the library powering it. Build your own tools on top of the same foundation.

CodeEditorLib

The core editing engine behind CE++. CodeEditorLib is an open-source .NET library providing the syntax-highlighting control, language definitions, and theme engine. It is available separately so developers can embed the same editing experience in their own Windows applications.

Getting Started

Up and running in minutes

CE++ is a standard Windows application. No dependencies required to start editing.

1

Download the latest release

Grab the latest .exe or .zip from the GitHub Releases page.

2

Run and Install

Run CE++_Setup.exe - Follow the installation process (it's not complicated) & Run.

3

Configure runtimes (optional)

To use the built-in code runner, open Run → Run Settings and point CE++ to your installed runtimes (Python, Node.js, g++, etc.), or leave empty to use the system PATH.

4

Start coding

Open a file with Ctrl + O, or create a new tab with Ctrl + N. CE++ auto-detects the language from the file extension.
