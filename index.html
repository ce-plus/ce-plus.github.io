CE++ is a fast, lightweight, and free code editor built for Windows. Designed for developers, students, and anyone who works with source code, CE++ delivers a clean and distraction-free editing experience without the heavy resource usage of larger IDEs.

What CE++ Does

CE++ is a multi-tab text and code editor that lets you open, edit, and save source code files across many programming languages - all in one simple window. Whether you're writing a Python script, editing an HTML page, or reviewing a SQL query, CE++ gives you the tools to do it comfortably.

Key Features

  • Multi-Tab EditingOpen multiple files at the same time and switch between them easily. Each tab shows whether a file has unsaved changes.
  • Syntax HighlightingCE++ automatically detects the programming language and highlights syntax, making code much easier to read.
  • Find & ReplaceQuickly locate any text, with options for case-sensitive search and replacing single or all occurrences at once.
  • Dark & Light ThemesSwitch between Dark and Light themes to match your working environment or personal preference.
  • Go To LineJump instantly to any line number in a large file without scrolling.
  • Toggle CommentComment or uncomment lines of code with a single shortcut, speeding up your workflow.
  • Recent FilesQuickly reopen files you've worked on before from the Recent Files menu.
  • Drag & DropDrag files directly from Windows Explorer into CE++ to open them instantly.
  • ZoomAdjust the text zoom level using Ctrl + Mouse Scroll for comfortable reading at any screen size.
  • Smart Encoding DetectionCE++ automatically detects the encoding of files you open (UTF-8, Unicode, etc.) so characters always display correctly.

About CodeEditorLib

The core editing engine powering CE++ is CodeEditorLib - an open-source library developed by the same author, Ari Sohandri Putra, released under the MIT License. Free to use, modify, and distribute. Developers are welcome to use it as the foundation for their own editors.
