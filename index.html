CE++ is a lightweight, open source code editor for Windows. It opens fast, runs quietly, and lets you focus on writing — not on managing the tool itself. No subscription. No account. Just an editor.

What you get
Everything an editor should do.

CE++ handles the things you'd expect from a modern editor — and a few things you might not expect from something this lightweight.

🗂️
Multi-Tab Editing
Work on several files at once without losing your place. Each tab shows a clear indicator when a file has unsaved changes.
🎨
Syntax Highlighting
CE++ recognizes the language automatically and colors your code accordingly, making structure easy to read at a glance.
🗺️
Code Minimap
A color-coded overview of your entire file sits on the right edge. Click or drag anywhere on it to jump to that position instantly.
🔍
Find & Replace
Locate any word or phrase in seconds. Replace one match or all at once, with options for exact case and whole-word matching.
↩️
Undo & Redo
Made a mistake? CE++ tracks up to 500 changes so you can step back as far as you need, without losing anything.
🎯
Go To Line
Jump to any line number in one step. The dialog shows a preview of the destination line and a visual slider for long files.
🕓
Recent Files
Your last-opened files are always one click away, so you can pick up where you left off without hunting through folders.
📂
Drag & Drop
Drag any file from Windows Explorer straight into CE++ to open it. No file dialog needed.
